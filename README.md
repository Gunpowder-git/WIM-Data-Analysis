# WIM Data Analysis | 动态称重数据分析

## 项目简介 | Project Overview

本项目基于某大桥 2024 年 3 月份的 WIM（Weigh-In-Motion，动态称重）数据，使用 Python 对车辆通行情况进行统计分析和可视化展示。

分析内容包括：总体车型占比、各车道车型分布、不同方向车流量对比、各车道车辆总重统计、单日小时车流变化，以及工作日与双休日车流规律对比。

This project analyzes WIM (Weigh-In-Motion) traffic data collected from a bridge in March 2024. Python is used to process the dataset, generate statistical summaries, and visualize traffic patterns.

The analysis includes overall vehicle type distribution, lane-level vehicle composition, directional traffic comparison, vehicle weight statistics by lane, hourly traffic flow for a selected full day, and optional weekday/weekend traffic pattern analysis.
