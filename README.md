# WIM-Data-Analysis
基于 WIM 动态称重数据的桥梁车辆通行分析项目，使用 Python 对车型分布、车道特征、方向车流、车辆重量和小时车流变化进行统计与可视化分析。 A Python-based WIM data analysis project for bridge traffic, including vehicle type distribution, lane-level statistics, directional traffic comparison, vehicle weight analysis, and hourly traffic flow visualization.
